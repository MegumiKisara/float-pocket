# 六、核心操作流程

1. 工具启动：电脑开机自启（若开启）或手动打开工具 → 工具后台静默运行，系统托盘显示图标。

3. OCR\&amp;翻译流程：Snipaste截图并复制图片 → 打开工具OCR功能页面 → Ctrl\+V粘贴图片或点击粘贴按钮 → 预览图片确认无误 → 选择“提取文字”或“一键翻译” → 查看结果 → 点击“一键复制”完成操作。

4. 快捷应用流程：打开快捷应用面板 → 切换单层分类（若启用分类） → 找到目标应用 → 点击启动，面板自动关闭。

5. 配置修改：托盘右键点击“打开设置面板” → 选择对应配置模块 → 调整参数（如API Key、分类、快捷键） → 自动保存，即时生效。

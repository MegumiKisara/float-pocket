# FloatPocket

Windows 轻量化悬浮效率工具。个人自用小项目。

## 功能

- **OCR / 翻译** — 截图后通过千问 API 识别文字或翻译
- **计划表** — 简单的待办事项管理
- **快捷应用** — 快速启动常用应用
- **全局快捷键** — Ctrl+Alt+S 调出悬浮球
- **系统托盘** — 托盘图标管理，支持开机自启

## 使用

下载 [Releases](https://github.com/k6684969-cmd/float-pocket/releases) 中的 `FloatPocket.exe`，直接运行。

首次使用需要在「设置」中配置千问 API Key（OCR/翻译功能需要）。


## 说明

- 纯个人工具，功能按自己需求来
